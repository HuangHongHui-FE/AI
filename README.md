## AI Agent

人工智能代理：利用大模型语言（LLM）的推理能力来规划观察和执行行动，从而追求并实现目标的系统；

LLM：大脑，本身不具有记忆

    规划：
        子目标分解：
        反思与改进：
    记忆：
        短期：上下文丢失，跨线程会丢失记忆
        长期：外部向量存储和快速检索（区分不同类型）
            语义记忆：存储常识，事实，概念，关系，构成人工智能关于世界的知识库。
            情景记忆：存储特定的过去事件和经历，记录人工智能互动的个人日记
            过程记忆：存储学习到的技能，程序。构成人工智能的行动库
    工具使用：
        调用外部API，调用专有源

管理长期记忆的关键决策：
    存储哪种类型的长期记忆
    如何存储和更新记忆
    如何检索记忆
    如何衰减记忆
