## monorepo

Monorepo（Monolithic Repository，单一代码仓库） 是一种项目管理模式，指将一个团队或公司的所有项目、模块、代码片段（无论是否关联）都存储在单个 Git（或其他版本控制系统）仓库中，而非为每个项目单独创建一个仓库（即传统的 Multirepo 模式）。
